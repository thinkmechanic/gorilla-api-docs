# Introduction

Welcome to the [Gorilla.io](https://gorilla.io/) API docs.

Getting yourself setup to use our API with your favorite programming language is
pretty easy.

```shell
# Install curl
apt-get install curl
```

```ruby
# Ruby Gems
gem install 'gorilla-io'
require 'gorilla-io'

# Gemfile
gem 'gorilla-io'
```
